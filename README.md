# My Learning Notes

A lightweight notes website that loads Markdown files from the `notes/` folder and displays them in a sidebar-based reader.

## Built With AI

This project was built with AI assistance using Codex.

## Project Structure

- `index.html` - app UI, theming, and markdown renderer
- `notes/` - markdown note files
- `notes/index.json` - note metadata used by the sidebar

## How to Use

1. Open `index.html` in a browser.
2. Add notes as `.md` files in `notes/`.
3. Register each note in `notes/index.json`.
